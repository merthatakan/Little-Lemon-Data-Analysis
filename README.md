# Little-Lemon-Data-Analysis
This repository contains a full-stack data solution for Little Lemon, a fictional Mediterranean restaurant. The project bridges the gap between database management and business intelligence by implementing a structured MySQL database and transforming raw data into actionable insights using Tableau.
